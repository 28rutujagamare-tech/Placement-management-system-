# Placement-management-system-
A Placement Management System is a web-based application built using Django that helps colleges, universities, and training institutes streamline their campus recruitment process. It acts as a bridge between students, placement officers, and recruiters, ensuring smooth communication and efficient management of placement-related activities

Purpose
The project aims to:
Reduce manual work in placement coordination.
Provide transparency between students and recruiters.
Maintain accurate records of placement history.
Improve efficiency of campus recruitment drives.

Technical Stack
Backend: Django (Python framework)
Frontend: HTML, CSS, Bootstrap (or React for advanced UI)
Database: SQLite/MySQL/PostgreSQL
Other Tools: Django ORM, Django Admin Panel

Functionalities
Authentication system (students, admin, recruiters).
Database-driven storage of student and company details.
Automated eligibility checking based on CGPA/skills.
Notifications for upcoming drives and results.
Dashboard with placement analytics (number of students placed, company participation, etc.).
